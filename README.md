# BlinkIT-Dashboard
# 📊 BlinkIT Grocery Sales Analysis Dashboard (Excel)
![ Dashboard ](https://github.com/nikitawarade11/BlinkIT-Dashboard/blob/main/bink1.png)

## 📌 Project Overview
This project presents an interactive **Excel dashboard analyzing BlinkIT grocery sales data** to uncover insights about product performance, outlet characteristics, and customer preferences. The dashboard enables quick analysis of sales trends, item categories, outlet sizes, and location performance using dynamic charts and pivot tables.

The goal of this project is to demonstrate **data cleaning, transformation, and visualization skills using Advanced Excel**.

---

## 📂 Dataset Information
The dataset contains grocery sales information with attributes such as:

- Item Fat Content
- Item Type
- Outlet Size
- Outlet Location Type
- Outlet Type
- Item Visibility
- Item Weight
- Sales
- Rating
- Outlet Establishment Year

These attributes help analyze how different factors impact product sales.
